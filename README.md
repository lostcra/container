# container
simple helloworld for containerization with docker

# preinstall
docker
git

# setup
how do you run this repository? just, consistently execute this instruction to launch your first container:
''' bash or powershell or cmd
git clone https://github.com/lostcra/container.git
cd container
docker build -t lostcra14/container:v1.0.0 .
docker run lostcra14/helloworld:v1.0.0
```
you can replace lostcra14/helloworld:v1.0.0 on your name:tag\
